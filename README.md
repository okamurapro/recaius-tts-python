# recaius-tts-python
このプロジェクトは、RECAIUSの音声合成APIの利用方法を理解するためのサンプルアプリケーションです。
このアプリケーションを動作させる場合は、事前に[RECAIUS ディベロッパー登録](https://developer.recaius.io/jp/top.html)が必要です。

## Requirements
Python 3が必要になります。
また、`pyaudio`のパッケージが必要になります。

```
pip3 install pyaudio
```


## Run
`tts.py`を実行してください。

```
python3 tts.py
```
